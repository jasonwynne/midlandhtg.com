== Description ==



Adding Bing Webmaster Tools verification code to your website has never been easier. Simply copy and paste your verification meta tag and that's it!



Why use Bing Webmaster Tools? 



* Analyze your website's performance on Bing search engine;

* Get ideas for improving the website;

* ...and more.



For more information visit: [Bing Webmaster Tools](http://www.google.com/webmasters) or [OnlineAds.lt](http://onlineads.lt).



== Installation ==



1. Upload `verify-bing-webmaster-tools` directory to the `/wp-content/plugins/` directory or use plugin search - Admin > Plugins > Add new > Search for 'Verify Bing Webmaster Tools'.

2. Activate the plugin through the 'Plugins' menu in WordPress 

3. Copy verification meta-tag from Bing Webmaster Tools and paste it in the settings page (Admin > Settings > Verify Bing Webmaster Tools).

4. Click 'Save' and it's done!
